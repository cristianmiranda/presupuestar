# FIAT - Presupuesto de Chapa y Pintura

## Run locally

```bash
# Install dependencies
bundle install

# Run rails server
rails server
```

## Deploy to fly.io

Dashboard: https://fly.io/apps/presupuestar

```bsah
fly deploy
```
